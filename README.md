ConsoleClient (beginner - intermediate)
--------------
This BB10 app shows how to connect an app to ConsoleDebug.

THIS APP DOESN'T NEED TO BE PART OF YOUR PROJECT, IT'S JUST AN EXAMPLE OF HOW TO CONNECT TO ConsoleDebug.
